# OC Web

**Note:** This is a namespace reservation package. The full `@kcrommett/oc-web` package will be released soon.

OC Web is a self-hosted web interface for the OpenCode AI assistant, rebranded from `opencode-web`.

For now, please use:
```bash
bunx opencode-web@latest
```

Stay tuned for the full `@kcrommett/oc-web` release!
