# CodeWeb

**Note:** This is a namespace reservation package. The full `codeweb` package will be released soon.

CodeWeb is a self-hosted web interface for the OpenCode AI assistant, rebranded from `opencode-web`.

For now, please use:
```bash
bunx opencode-web@latest
```

Stay tuned for the full `codeweb` release!
